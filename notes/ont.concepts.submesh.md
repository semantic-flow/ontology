A submesh is a mesh that is embedded within another mesh.

It is not a separate fundamental kind from `SemanticMesh`; "submesh" names the role a mesh plays when another mesh includes or integrates it.

This should stay independent of any one implementation technique. A submesh might be realized through a Git submodule, a vendored nested directory, generated output, or some other embedding mechanism.
